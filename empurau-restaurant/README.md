# EMPURAU Restaurant Website

Website profesional, modern, dan elegan untuk EMPURAU Restaurant - restoran seafood premium di Jakarta Barat.

## Deskripsi

Website ini dirancang untuk menampilkan EMPURAU Restaurant sebagai destinasi kuliner seafood berkualitas tinggi dengan desain yang elegan dan responsif.

## Fitur

- **Beranda**: Hero section dengan CTA buttons
- **Tentang Kami**: Cerita restoran dan fitur unggulan
- **Menu**: Tampilan menu seafood dengan harga
- **Ulasan**: Rating dan testimoni pelanggan
- **Lokasi & Kontak**: Peta, alamat, jam operasional, dan informasi kontak
- **Footer**: Informasi restoran dan link sosial media

## Teknologi

- HTML5
- CSS3 (dengan Bootstrap 5)
- JavaScript (ES6+)
- Font Awesome untuk ikon
- Google Fonts untuk tipografi

## Setup

1. Pastikan web server (seperti Apache/Nginx) sudah terinstall
2. Clone atau download file-file website ke direktori web server
3. Buka `index.html` di browser

## Gambar yang Dibutuhkan

Website ini memerlukan beberapa gambar untuk tampilan yang optimal. Letakkan gambar-gambar berikut di folder `assets/images/`:

### Gambar Wajib:

- `hero-seafood.jpg` - Gambar hero section (seafood premium, ukuran besar)
- `restaurant-interior.jpg` - Interior restoran yang elegan
- `seafood-platter.jpg` - Platter seafood untuk menu
- `grilled-fish.jpg` - Ikan bakar spesial
- `seafood-soup.jpg` - Sup seafood

### Rekomendasi:

- Resolusi minimal: 1920x1080 untuk hero, 800x600 untuk menu
- Format: JPG atau PNG
- Tema: Elegan, premium, fokus pada seafood

## Kustomisasi

### Mengubah Informasi Restoran:

Edit bagian yang relevan di `index.html`:

- Alamat, telepon, jam operasional
- Deskripsi menu dan harga
- Testimoni pelanggan

### Mengubah Styling:

Edit `assets/css/style.css` untuk:

- Mengubah warna tema
- Menyesuaikan layout
- Menambah animasi

### Menambah Fungsionalitas:

Edit `assets/js/main.js` untuk:

- Formulir reservasi
- Integrasi Google Maps yang lebih akurat
- Animasi tambahan
